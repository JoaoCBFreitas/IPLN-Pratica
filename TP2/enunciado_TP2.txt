Para o Trabalho Prático 2 cada um dos grupos deverá escolher 1 dos seguintes temas:

1) Dicionário de rimas
    Desenvolver um programa com as seguintes funcionalidades:
        - Dada uma palavra, apresentar uma lista de palavras que rimam com ela. Esta funcionalidade deve ter uma opção que permita apresentar qualquer tipo de palavras (não relacionadas com a palavra de input), ou apresentar apenas palavras que tenham alguma relação semântica/contextual com a palavra de input;
        - Dado um ficheiro de um poema, anotar os versos que rimam entre si.
    Para cumprir esta tarefa, deve ser feita uma recolha de poemas (usando scraping) que devem ser processados de forma a que todos sigam uma estrutura definida.
    Por exemplo, representar cada poema num ficheiro JSON que tenha as seguintes informações:
        Título, Autor, Data, Obra (se pertencer a uma obra com vários poemas), Tema, Parágrafos (o poema propriamente dito)
